# Avito_QA_internship_tasks
Задания для стажировки на QA в авито

**qa_internship_tests**

Автоматизированные pytest тесты для API сервиса объявлений.

**Требования**
- Python 3.8+
- pip

**Установка**
1. Клонировать/скопировать проект
2. Создать виртуальное окружение (рекомендуется)
   python -m venv .venv
   source .venv/bin/activate   # mac/linux
   .\.venv\Scripts\activate    # windows

3. Установить зависимости:
   pip install -r requirements.txt

**Запуск тестов**
- Все тесты:
  pytest -v

- Конкретный тест:
  pytest tests_task2/test_api.py::TEST_NAME -q

**Примечания**
- Тесты работают против публичного хоста `https://qa-internship.avito.com` (из Postman коллекции).
- В тестах используются случайные sellerID в диапазоне 111111-999999, для минимизации пересечения.
- Тесты могут зависеть от поведения тестовой среды (некоторые негативные кейсы ожидают 400/422 и т.д.). В случае нестабильности сервиса, нужно скорректировать код проверок.
